    - Khởi tạo PW: mở terminal gõ "npm init playwright@latest"
    - Các thư mục:
    + node_modules: chứa thư viện, cài thêm thư viên
    + tests - example.spec.ts: code test
    + playwright.confi.ts: file cấu hình (chụp ảnh, video..)
    



    - Đưa code lên git:
+ Làm 1 lần duy nhất:
        ○ Khởi tạo repo local: git init
        ○ Tạo repo và liên kết tới repo local: git remote add origin <url>
    + Làm mỗi khi có thay đổi:
        ○ Thêm file vào stagging: it add .
        ○ Commit file: git commit -m "message"
        ○ Push code lên repo remote: git push origin main
